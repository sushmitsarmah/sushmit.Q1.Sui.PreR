# zkSend Integration

DApp Kit provides out-of-the-box opt-in support for [zkSend wallets](../zksend/dapp.mdx).

## Setup

To enable support for zkSend wallets, you need to pass the `zkSend` object to the `WalletProvider`
component. This object has the following properties:

- **`name`** - The name of your dApp. This will be shown to the user when connecting to the dApp.

```tsx
function App({ children }) {
	return (
		<WalletProvider
			zkSend={{
				name: 'Your dApp name',
			}}
		>
			{children}
		</WalletProvider>
	);
}
```
